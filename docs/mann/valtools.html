DESCRIPTION

valtools(n) contains commands to aid in the validation of user input.

COMMANDS

prepare varname ?options...?

Prepares the content of variable varname for further validation. First, the content is trimmed of leading and trailing whitespace. Then, the options are applied in order. Some options transform the value; others can begin to validate it. The latter throw INVALID on error.

The options are as follows:

-tolower
The variable's value is converted to lower case.
-toupper
The variable's value is converted to upper case.
-required
Throws INVALID if the value is the empty string.

ENVIRONMENT

Requires Tcl 8.6 or later.
