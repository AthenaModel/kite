SYNOPSIS

package require kiteutils 0.5.0
namespace import ::kiteutils::*
  • pluck file fname url

DESCRIPTION

pluck(n) contains commands to retrieve resources from HTTP servers.

HTTPS URLs

To retrieve files from "https://" URLs, follow the instructions on the http(n) man page. In particular:

  • Require the "tls" package in project.kite
  • package require tls in the application's pkgModules.tcl

  • In the main body of the application, package require http, and then: 

    ::http::register https 443 ::tls::socket

COMMANDS

pluck file fname url

Attempts to retrieve the document at the url and save it to disk as fname, throwing NOTFOUND on error.

The url may be any "http://" URL accessible from the user's host. If the tls(n) package is loaded and registered with the http(n) package, then "https://" URLs can be used as well.

ENVIRONMENT

Requires Tcl 8.6 or later.
