DESCRIPTION

control(n) contains commands the modify the flow of control in some way.

COMMANDS

assert expression

assert tests an invariant Boolean expression. If the expression is false, assert throws the following error: 

Assertion failed: expression

The Tcl errorCode variable will be set to "ASSERT".

bgcatch script
Executes an arbitrary Tcl script in the caller's context. If the script throws an error, the error is handled by bgerror. bgcatch never returns a value.
callwith prefix ?args...?
Calls a command prefix with zero or more additional arguments in the global scope. This is to be used when calling callback commands. If prefix is the empty list, then callwith does and returns nothing.
foroption optvar argvar ?-all? body

The foroption command is a tool for looping over options and values from an argument list. Here, argvar is the name of the variable containing the argument list, and optvar is the name of a variable to receive the successive option names. The body is a set of Tcl switch cases, one for each valid option.

The options are extracted from the argument list using lshift. When an option takes a value, the relevant switch case can extract its value from the list in the same way.

It's an INVALID error if foroption reads an option without a matching case.

By default, foroption keeps reading options until it reaches the end of the argument list, or finds a list item that begins with something other than "-". In this case, the argvar will contain the non-option arguments when foroption returns.

If the -all option is included, foroption throws an INVALID error if finds an argument that isn't an option or an option value.

For example, 

proc myproc {args} {
    array set opts {
        -this {}
        -that {}
    }

    foroption opt args -all {
        -this -
        -that { 
            set opts($opt) [lshift args] 
        }
    }

    ...
}
require expression message
This is identical to assert except that error message is the specified message.

ENVIRONMENT

Requires Tcl 8.6 or later.
