DESCRIPTION

stringx(n) contains a collection of string-manipulation utilities.

COMMANDS

codeblock name arglist ?initbody? template

Defines a command called name in the caller's context; the new command will return the block of code contained in the template after making several transformations.

First, the template will be outdented using outdent. Second, double backslashes will be replaced with single backslashes; this allows the generated code block to contain continuation backslashes.

Finally, string map will be called to replace the block's arguments with the values in the caller's arglist. Argument names appear in the template prefixed by a percent sign, "%". For example, 

% codeblock hello {first last} {
    Hello, %first %last!
    How are you?
}
::hello
% hello John Doe
Hello, John Doe!
How are you?
%

If given, the initbody may transform the arguments in some way or define additional scalar local variables for use in the template: 

codeblock timestamp {seconds} {
    set date [clock format -fmt "%D"       $seconds]
    set time [clock format -fmt "%H:%M:%S" $seconds]
} {
    Timestamp: at %time on %date
}

The codeblock command is similar to template(n), but is better suited to creating templates of Tcl code. It can also be used for creating arbitrary strings.

normalize text
Given a text string, trims leading and trailing whitespace, and reduces all internal whitespace to single spaces.
outdent block
Given a block of text, "outdent" deletes any leading blank line or trailing blank line. It also determines the longest common whitespace prefix shared by the remaining lines (assuming the whitespace does not contain tab characters) and deletes it from each line.

ENVIRONMENT

Requires Tcl 8.6 or later.
