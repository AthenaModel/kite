DESCRIPTION

manpage(n) defines a formatter for project man pages written in manpage(5) format. manpage(5) is an "Extended HTML" format; a man page is an HTML-like file with embedded macros that make it easy to adhere to man page format.

See manpage(5) for a description of man page syntax and the available macros. Note that individual man pages can define their own macros.

Cross-References

manpage(n) supports ehtml(5)'s xref macro. It is assumed that the man page directories are direct children of the project's docs/ directory.

COMMANDS

This module provides the following commands:

manpage register macroset
Registers a macroset(i) macro set with manpage(n). The macro set will be available at the next and subsequent calls to format.
manpage format srcdir destdir ?options...?

Given the name of a directory containing manpage(5) files (the srcdir), this command produces an HTML man page for each, along with an HTML "index.html" file, and puts them in the destination directory (destdir).

Note that the macro expander is reset between man pages; page-specific macros do not carry over from one man page to the next.

-project project
Specifies the name of the project being documented. Defaults to "YourProject". This string appears in the headers and footers of the generated pages.
-version x.y.z
Specifies the client project's version number, for inclusion in the generated man pages. Defaults to "0.0.0", as a placeholder.
-description description
Specifies a brief one-line description of the project being documented. Defaults to "Your project description". This string appears in the headers of the generated pages.
-section title
Specifies the section title for the collection of man pages in the srcdir. This section title will be used in the generated index page.
-errfile filename
The format command processes each manpage(5) files in several passes. If an error is detected in a particular pass, the intermediate input to that pass is written to the given filename as an aid to diagnosing the problem.

ENVIRONMENT

manpage(n) requires Tcl 8.6 or later.
